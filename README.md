# Calculadora de Índice de Masa Corporal (IMC)

## Descripción

Este programa solicita al usuario:

- Nombre
- Apellido paterno
- Apellido materno
- Edad
- Peso
- Estatura

Posteriormente valida que ningún campo quede vacío y que la edad, peso y estatura sean datos numéricos válidos.

Después calcula el Índice de Masa Corporal (IMC) mediante la fórmula:

IMC = peso / (estatura ** 2)

Finalmente muestra todos los datos ingresados por el usuario junto con el resultado del IMC.

## Cómo ejecutar el programa

1. Descargar el archivo calculadora_imc.py.
2. Abrir una terminal o consola.
3. Ejecutar el siguiente comando:

python calculadora_imc.py
